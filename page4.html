This project was a game Nathanael and I made in 10th grade called "Super Smash Boot," which is essentially a recreation of the video game series "Super Smash Bros." This fighting-game project was created from scratch and we had roughly 2 weeks - 1 month to create this project. This project was made with MelonJS, while also implementing PHP elements as well. We wanted to recreate this series since it was a series we both enjoyed and still play sometimes to this day. Similar to the APUSH project, I enjoyed working on the project everyday and loved seeingthe project come together. The most frustrating part of this porject was working with spritesheets we found online, but we solved it using a program called Piskel. Granted, this is an old project created in 10th grade, when I only had 1 and a bit more years of programming experience under my belt, so the code and styling might be a little off.
