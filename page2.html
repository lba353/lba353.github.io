The project is a p5.js game called "Coin Collector." I paired up with a close friend of mine (Nathanael Chow) to create a game that only had two main goals: is interactive and has a scoring system. The major JavaScript concepts we used were the p5 Library; which included p5, p5play, p5sound, and p5dom. We initially designed it to be a flat grassland where your only goal is to collect coins within the time limit. After we presented the first version of the project, we added more elements to the project. To play the game, you use the left or right arrow keys to move and the up arrow key to jump.
